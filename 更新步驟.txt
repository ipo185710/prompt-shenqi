提示詞神器改成免登入版

1. 下載並解壓縮本 ZIP。
2. 到 GitHub 的 prompt-shenqi Repository。
3. 刪除舊檔案：
   app.js
   config.js
   schema.sql
   SETUP.md
   styles.css
   index.html
   vercel.json
4. 點 Add file → Upload files。
5. 上傳新版資料夾裡的：
   index.html
   vercel.json
   README.md
6. 按 Commit changes。
7. 等待約 30～60 秒，Vercel 會自動重新部署。
8. 重新開啟：
   https://prompt-shenqi.vercel.app

注意：
資料保存在每個使用者自己的瀏覽器中；換裝置或清除瀏覽器資料前，請先使用網站的「匯出備份 JSON」。
